![](styles.csv)

## Stable Diffusion 思路

基本思路：Goal -> Style -> ZeroShot/Few Shot -> Negative Prompt

1. Goal：
	- a photo,
	- a painting,
	- a sculpture,
	- a 3D rendering, etc. 
2. Style：诸如于动漫（Anime）、半写实、写实、真人等等
3. 开始模式：ZeroShot/FewShot，以获得更多的灵感
4. 精炼 Prompt
     - 反复尝试 Prompt
     - 添加 Negative Prompt
     - 使用 seed 重新运行
